### Nanopore-caller: alternative basecaller for MinION reads

Requirements
================

We use Python 2.7.

Here are versions of Python packages, that we used:

- Cython==0.23.4
- numpy==1.10.2
- h5py==2.5.0
- Theano==0.7.0
- python-dateutil==2.5.0

Basic usage:

`python basecall.py <list of fast5 files>`
